##  ✨Project Overview

### **사용기술**

- HTML5 / CSS3
- JavaScript(ES6+)
- React
- Redux, Redux-toolkit
- SCSS
- Ant design

### **구현기능**

- 로그인/로그아웃
- 랜딩페이지 프로덕트 무한스크롤
- 상품 검색 (추천 키워드 : watch, top, ... 영문 키워드 지원)
- 카테고리 분류
- 제품 상세 페이지
- 마이페이지 (주문내역은 dummyjson API 활용)
- 장바구니 추가, 삭제
- 반응형
